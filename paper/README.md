# Paper: The Asian Language Tax

LaTeX source for the arXiv-ready manuscript. Modeled on Somide (2026), *The African Language Tax* (arXiv:2606.24460).

## Build

```bash
cd paper
latexmk -pdf paper.tex
# or upload paper.tex + refs.bib + figures/ to Overleaf
```

Figures are pulled from `../runs/main/figures/`. The PDF version of each figure is automatically generated by `latexmk` from the PNG sources via `graphicx`.

## Files

- `paper.tex` — main manuscript
- `refs.bib` — bibliography (natbib + plainnat)
- (Figures live in `../runs/main/figures/` — six PNG + SVG pairs)

## Overleaf

Upload `paper.tex`, `refs.bib`, and the six PNG figures from `runs/main/figures/`. Compile with pdfLaTeX.

## arXiv submission

1. Run `make build` (or `latexmk -pdf`) to produce `paper.pdf`.
2. Zip `paper.tex`, `refs.bib`, and all referenced PNG figures into one archive.
3. Submit at https://arxiv.org/submit with primary class `cs.CL`.
4. Use the abstract from `paper.tex` as the submission's abstract.
5. Title: *The Asian Language Tax: Quantifying the Cost, Context, and Recall Penalty of Tokenizing Lower-Resource Asian Languages in Frontier LLMs*
